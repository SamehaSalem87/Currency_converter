# Currency Converter 

Ein einfaches, benutzerfreundliches **Webprojekt**, mit dem Nutzer verschiedene Währungen in Echtzeit umrechnen können.  
Dieses Projekt wurde mit **HTML, CSS und JavaScript** umgesetzt und ist als **Web-App** über GitHub Pages verfügbar.

---

## Live-Demo

[Hier ausprobieren](https://samehasalem87.github.io/Currency_converter/)

---

##  Technologien

- **HTML5** – Struktur der Webseite  
- **CSS3** – Styling und Layout  
- **JavaScript** – Logik für die Umrechnung und dynamische Interaktionen  
- **GitHub Pages** – Hosting der Web-App

---

##  Features

- Umrechnung zwischen **verschiedenen Währungen** in Echtzeit  
- **Intuitive Benutzeroberfläche** für schnelle Eingaben  
- **Automatische Aktualisierung der Ergebnisse** bei Eingabeänderung  
- **Responsives Design** – funktioniert auf Desktop und Mobilgeräten  
- Leicht erweiterbar, um weitere Währungen hinzuzufügen

---

##  Verwendung

1. Öffne die Webseite: [Currency Converter](https://samehasalem87.github.io/Currency_converter/)  
2. Wähle die **Quellwährung** und die **Zielwährung** aus  
3. Gib den **Betrag** ein, der umgerechnet werden soll  
4. Das Ergebnis wird **sofort angezeigt**
